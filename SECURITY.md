# Security policy

## Reporting a vulnerability

If you believe you have found a security vulnerability in this project:

- Please **do not** open a public issue with full details.
- Prefer using **GitHub Security Advisories** (private disclosure) if this repository is hosted on GitHub.

If private reporting is not available, open an issue with the title prefix **"[SECURITY]"** and provide only
high-level details (impact + affected component). A maintainer can follow up for details.

## Supported versions

This is research code. Security fixes may be provided on a best-effort basis for the latest release/tag.
